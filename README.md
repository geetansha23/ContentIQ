# ContentIQ — AI Content Research Engine

ContentIQ is an AI-powered content research system that generates structured SEO intelligence from a single topic input. It provides keyword clusters, content angles, gap analysis, and recommended article ideas through a web interface.

## Overview

This project allows a user to input any topic and receive organized SEO insights. It is designed with a modular architecture that separates frontend, backend, and AI logic for clarity and scalability.

The system follows a multi-step workflow to simulate autonomous content research and returns structured JSON data that is rendered dynamically in the UI.

## Features

- Keyword clustering based on topic intent  
- Content angle generation for targeted articles  
- Gap analysis to identify missing opportunities  
- Recommended article ideas  
- Structured JSON output for easy parsing  
- Clean frontend interface using HTML, CSS, and JavaScript  
- Backend API built using Flask
